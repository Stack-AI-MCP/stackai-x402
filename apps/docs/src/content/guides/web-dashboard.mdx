# Web Dashboard

The stackai-x402 web dashboard provides a visual interface for discovering agents, monetizing servers, chatting with tools, and tracking analytics.

<Callout type="info">
  The live dashboard is at **[x402.stacks-ai.app](https://x402.stacks-ai.app)**. Connect your Leather or Xverse wallet to get started.
</Callout>

## Marketplace

Browse all registered agents and MCP servers.

<ThemeImage lightSrc="/images/marketplace-light.png" darkSrc="/images/marketplace-dark.png" alt="Marketplace — browse all registered MCP servers" caption="Browse agents with descriptions, tool counts, and pricing" />

- **Browse** — View all registered agents with descriptions, tool counts, and pricing
- **Filter by token** — Filter servers by accepted payment token (STX, sBTC, USDCx)
- **Details** — Click an agent to see its full configuration, system prompt, and available tools
- **Chat** — Launch a terminal session directly with any agent

## Register (Monetize)

Register your MCP server to start earning from tool calls.

<ThemeImage lightSrc="/images/register-light.png" darkSrc="/images/register-dark.png" alt="Register — monetize your MCP server" caption="Enter your server URL, set pricing, and register in minutes" />

### Steps

1. **Connect Wallet** — Use Leather or Xverse to connect your Stacks wallet
2. **Enter Server URL** — Provide your MCP server's HTTP endpoint
3. **Introspect** — The gateway discovers all available tools from your server
4. **Set Pricing** — Configure USD prices per tool (or $0 for free tools)
5. **Agent Profile** — Add a name, description, system prompt, and starter prompts
6. **Register** — Your server is now monetized and discoverable

<Callout type="info">
  When you register, your connected wallet address becomes the **owner**. All tool payments go directly to this address on-chain.
</Callout>

### Accepted Tokens

During registration, select which tokens your agent accepts:
- **STX** — Native Stacks token
- **sBTC** — Wrapped Bitcoin on Stacks
- **USDCx** — Circle's USDC bridged to Stacks

## Chat Terminal

Test any agent interactively in the browser.

First, select an MCP server from the chat selector:

<ThemeImage lightSrc="/images/chat-select-light.png" darkSrc="/images/chat-select-dark.png" alt="Chat — select an MCP server" caption="Pick any registered agent to start chatting" />

Then chat with the server's tools using natural language:

<ThemeImage lightSrc="/images/chat-interface-light.png" darkSrc="/images/chat-interface-dark.png" alt="Chat — interactive terminal with starter prompts" caption="Natural language chat with tool execution and payment prompts" />

### Features

- **Message Streaming** — Real-time AI responses via SSE
- **Tool Execution** — See tool calls and results inline
- **Payment Prompts** — When a premium tool is called, a payment card appears with:
  - Tool name, price, and accepted tokens
  - Token selector (STX / sBTC / USDCx)
  - Approve / Reject buttons
  - Transaction status and hash after settlement
- **Model Selector** — Choose between different LLM models (Claude, GPT-4, etc.)
- **Markdown Rendering** — Assistant responses render with full markdown support

### Payment Flow in Chat

1. You send a message that triggers a premium tool
2. The AI calls the tool → a **PaymentCard** appears
3. Select your preferred token and click **Approve**
4. Your wallet prompts for transaction signing
5. The transaction is broadcast and verified
6. The tool result appears in the chat

</Callout>

## Wallet Integration

The dashboard uses [Stacks Connect](https://docs.stacks.co/stacks-connect/connect-wallet) for wallet integration:

- **Leather** and **Xverse** wallets supported
- Wallet is used for:
  - Authentication (message signing challenge)
  - Transaction signing (tool payments)
  - Owner verification (agent management)
- **No private keys** are ever stored in the browser — all signing happens in the wallet extension
