## Roles

stackai-x402 has two primary roles:

- **Consumer** — An AI agent (or developer building one) that calls premium tools and pays for them automatically using a private key
- **Provider** — A developer who runs an MCP server and wants to charge for tool usage

You can be both — the SDK supports consumer and provider workflows from the same package.
